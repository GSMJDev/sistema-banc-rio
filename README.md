💰 Sistema Bancário
Um sistema bancário simples desenvolvido em Python para gerenciamento de contas, depósitos e saques.

🚀 Funcionalidades
✅ Criar contas bancárias
✅ Depositar dinheiro
✅ Sacar dinheiro
✅ Consultar saldo

🛠️ Tecnologias
Python
JSON (para armazenamento de dados)
